右クリックで呼び出せる データ管理ツール ファイルのバージョン差分を簡単に作成できます。
アーカイブモードと増分保存モード2種類があります。


※保存前のデータを複製して保管する為のツールです。 新規保存、上書き保存は各自で行ってください。


※使用する前に

python 2.6以上がインストールされている必要があります。
環境変数にpythonパスが通っている必要があります。


インストール

zipを解凍してINSTALL.EXEを実行してください。



使い方

ファイルを選択して右クリックメニューから実行できます。


コマンド-選択したファイルをアーカイブする

ファイルを選択した状態で[選択したファイルをアーカイブする]を実行すると
ファイルのある場所に/_archive/ファイルの名前のフォルダを作成します。
-用途-
ファイルのバージョン管理にお使いください。
修正前データを保存等
-実際の挙動-
<選択ファイル名>.ext(拡張子)
_archive/<選択ファイル名>/<選択ファイル名>_v0001.ext
_archive/<選択ファイル名>/<選択ファイル名>_v0002.ext
・
・
・



コマンド-選択したファイルを増分保存する

ファイルを選択した状態で[選択したファイルを増分保存する]を実行すると
ファイルのある場所に/_archive/ファイルの名前のフォルダを作成します。
この挙動はMayaの Save Scene Options の Incremental save をオンにした時と同じです
-用途-
通常のファイルの保存としてお使いください。


-実際の挙動-

<選択ファイル名>.ext(拡張子)
incrementalSave/<選択ファイル名>/<選択ファイル名>.0001.ext
incrementalSave/<選択ファイル名>/<選択ファイル名>.0002.ext
・
・
・
